Overview

Executive Summary

VisionTrack is a self-hosted AI video analytics platform built to run entirely on-premises. Organisations upload CCTV footage through a browser interface and receive automated object detection reports, zone-level heat maps, and annotated playback — without a single frame leaving the local network.

🎯

AI Object Detection

YOLOv8-powered detection with per-frame bounding boxes, class labels, and confidence scores. Supports 80+ object categories out of the box.

🔗

Multi-Object Tracking

ByteTrack assigns persistent track IDs across frames, enabling complete journey tracing of every detected object through the scene.

📊

Automated Analytics Reports

Six-sheet Excel workbook generated after each analysis run — zone summaries, class breakdowns, per-minute timelines, and embedded charts.

▶️

Annotated Video Playback

H.264 annotated video with all bounding boxes and track IDs rendered — streamed directly in the browser with full seek support.

🔒

100% On-Premise

Runs entirely on local hardware. No footage, metadata, or results are transmitted to any external server. Fully air-gap compatible.

⚡

Axelera AI Acceleration

Native support for the Axelera M.2 AI card (Metis) via Voyager SDK — hardware-accelerated inference for real-time throughput on edge hardware.

Excel Report

Six-Sheet Analytics Workbook

Each completed job produces a colour-coded, chart-embedded Excel report ready for security audits, management briefings, or forensic review.

Sheet Colour Contents
Summary Blue File info, video duration, resolution, FPS, total detections, unique tracked objects, peak activity minute, model used, confidence threshold
Zone Summary Green Detection count and unique object count per configured zone — sorted by frequency. Includes embedded bar chart.
Class Breakdown Amber Per object class: detection count, unique objects, min/max/average confidence. Includes embedded bar chart.
Timeline Purple Per-minute rows: detection count, active objects, top 3 classes present. Peak minute highlighted. Includes embedded line chart.
Tracks Red Per tracked object: class, zone, first seen, last seen, dwell time, detection count, average confidence
Raw Detections Teal Frame-level data: frame number, timestamp, track ID, class, zone, confidence, bounding box coordinates (x, y, w, h) — up to 2,000 rows
Zone Intelligence

Configurable Detection Zones

Define custom polygon zones — entrances, car parks, restricted areas — and the system automatically classifies every detection into the correct zone using a ray-casting algorithm.

📐

Custom Polygon Zones

Define any number of named zones as pixel-coordinate polygons. Any shape, any size — not limited to rectangles.

🗺️

Auto Quadrant Fallback

If no zones are configured the system automatically divides the frame into four quadrants (NW, NE, SW, SE) for baseline spatial reporting.

📈

Zone Frequency Analysis

The Zone Summary sheet ranks areas by total detections and unique objects — instantly revealing the busiest locations in each clip.

⏱️

Dwell Time Tracking

The Tracks sheet records each object's first and last appearance times, enabling dwell time calculation per zone and per object class.

Deployment

Deployment Model

VisionTrack runs entirely on a single server. No internet connection, cloud account, or subscription service is required after initial setup.

🖥️

Recommended Hardware

Lenovo ThinkStation P360 Ultra (or equivalent). Ubuntu 22.04 LTS. Axelera AI M.2 card for hardware inference. 16 GB RAM minimum.

🌐

Browser Access

Accessed from any device on the local network via a standard web browser — no client software installation required.

📂

Supported Formats

Accepts MP4, AVI, MOV, MKV, and WebM video files up to 2 GB per upload. Output is always H.264 MP4 for maximum browser compatibility.

🔄

Non-Blocking Processing

Jobs run asynchronously — multiple uploads can be queued and the browser polls for status updates every 2.5 seconds automatically.
